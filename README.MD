#  To-Do List App

This is a simple To-Do List web application that I built to practice JavaScript and DOM manipulation. It allows users to add and remove tasks easily with a clean and user-friendly interface.

##  Features
- Add new tasks
- Delete tasks instantly
- Simple and clean UI
- Responsive layout (works on different screen sizes)

##  Technologies Used
- HTML for structure  
- CSS for styling  
- JavaScript for functionality  

##  Project Structure
- index.html – main structure of the app  
- style.css – styling and layout  
- script.js – logic for adding and deleting tasks  

##  Purpose of this Project
I created this project to improve my understanding of JavaScript, especially how DOM manipulation works in real-world applications. It helped me learn how to handle user input and dynamically update the UI.
